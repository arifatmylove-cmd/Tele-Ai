# 🚀 Pentest Telegram AI Bot

**Unrestricted pentesting assistant with Shodan + APK generation**

## 🚀 2-Minute Deploy

1. **Fork/Clone** this repo
2. **Create bot:** `@BotFather` → `/newbot`
3. **Get Chat ID:** Message bot `/getid`
4. **Deploy Render:**
   - Connect GitHub
   - Add `TELEGRAM_TOKEN`
   - Add `ADMIN_CHAT_ID` (your chat ID)
   - Deploy! ✅
